News

Timereports endpoint docs update and new field addition - 25.08.21

/timereports endpoints updated to more readable model (with JSON schema now via download link) along with new property invoice in the /timereports GET.

Invoices endpoint docs update and new field addition - 02.07.2021

/invoices endpoint properties list updated to more readable model (with JSON schema now via download link).

New properties available in the GET (list): customer, customerName, InvoicedAmountVAT, creditedInvoice and currencyCode.

Portfolio api available now - 12.11.2020

/portfolios endpoint has has been created for basic operations like post, post by id , get all and get by id.

We will be progressively upgrading all endpoints to list more readable properties and specific to each endpoint and JSON schema will be downloadable from the link in each

Projects schema fixes and new model for API docs - 28.10.2020

/projects endpoint has a minor fix as a part of the schema update to the new model for API docs: totalCost > forecastTotalCost. Along with this change, budgetedTotalCost (previously missing) has been added to /projects.

accounts enhancements and more additions - 05.10.2020

/accounts endpoint has a minor fix for consistency with all other endpoint to support our new model for API docs: eInvoiceAddress > eInvoicingAddress.
